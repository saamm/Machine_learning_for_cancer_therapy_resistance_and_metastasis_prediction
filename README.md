## Overview

This project tests the hypothesis that **therapy resistance and metastatic potential share underlying transcriptional states**. We train a variational autoencoder on TCGA pan-cancer expression data with a pharmacogenomic contrastive objective derived from GDSC2 drug sensitivity profiles, then evaluate whether the learned embeddings predict metastatic outcome without any direct metastasis supervision.

### Key Results

| Metric | Value |
|--------|-------|
| Metastasis prediction AUC (Random Forest) | **0.697** |
| Average precision | **0.817** |
| Survival stratification log-rank p | **1.6 × 10⁻¹¹⁵** |
| Cancer types with AUC ≥ 0.60 | **13 / 33** |

### Biological Finding

Two latent dimensions drive metastasis prediction:
- **z25** — PI3K / mTOR / stemness axis (MTOR, NANOG, AKT2, TSC2)
- **z3** — EMT / matrix remodelling / mitotic instability (MMP2, MMP9, FN1, TGFB1, AURKA)

These are known drivers of both therapy resistance and metastasis, recovered without supervision.

## Architecture

```
TCGA expression (97 hallmark genes)          GDSC2 cell lines
        │                                           │
   VAE Encoder                            Contrastive pairs
   (MLP + LayerNorm)                  (same resistance class = positive)
        │                                           │
   z ~ N(μ, σ)  ←────── shared latent space ──────→
        │
   VAE Decoder
        │
Reconstructed expression

Loss = Reconstruction + β·KL + γ·Contrastive
```

## Methods Summary

### Gene Panel

97 cancer-relevant genes covering: EMT (VIM, CDH1, ZEB1, SNAI1), PI3K/AKT/mTOR signalling, RAS/MAPK, p53/DNA damage response, immune evasion (CD274, TGFB1), stemness (SOX2, NANOG, CD44), apoptosis (BCL2, MCL1), and cell cycle regulators.

### Representation Learning

A VAE with two hidden layers (97 → 64 → 32 latent dimensions) is trained on 11,060 TCGA tumours with a reconstruction + KL objective (β = 0.5). Simultaneously, GDSC2 cell lines are passed through the same encoder with a contrastive loss (γ = 0.3) where positive pairs share the same resistance classification for a given drug pathway (top/bottom tertile of mean IC50).

### Resistance Labels

For each of 23 GDSC2 drug pathway groups, cell lines in the top tertile of mean pathway IC50 are labelled *resistant*, bottom tertile *sensitive*, and middle excluded from contrastive training.

### Metastasis Prediction

VAE embeddings (μ, deterministic) for 10,091 TCGA patients with known tumour status are used as features for 5-fold stratified cross-validated classifiers. Performance reported as ROC-AUC and average precision.

## Results

### UMAP of Resistance-Aware Latent Space

The latent space recovers cancer type structure without tissue-of-origin supervision, with transcriptionally distinct cancers (TGCT, KIRC, LAML) forming isolated clusters and epithelial tumours (BRCA, LUAD, BLCA) co-localising in a shared manifold.

### Metastasis Prediction

| Model | AUC | Average Precision |
|-------|-----|-------------------|
| Logistic Regression | 0.662 | 0.795 |
| Random Forest | **0.697** | **0.817** |
| Gradient Boosting | 0.690 | 0.812 |

### Survival Stratification

Patients stratified by predicted metastatic risk show highly significant differences in overall survival (log-rank p = 1.6 × 10⁻¹¹⁵), with 13/33 cancer types achieving per-type AUC ≥ 0.60.

## Limitations and Future Work

- **Gene panel scope:** 97 hallmark genes limit discriminative capacity. Genome-wide features would likely improve AUC substantially.
- **Tumour status proxy:** `tumor_status` is an imperfect metastasis label; prospective metastasis-annotated cohorts (e.g. MSK-MET with RNA-seq) would strengthen validation.
- **Cell line to patient transfer:** The contrastive signal comes from cell lines; domain adaptation methods could better bridge in vitro pharmacology to patient biology.
- **Scale:** This framework is designed to scale to whole-genome sequencing, whole-transcriptome, and digital histopathology — the natural next step within a PhD programme.
